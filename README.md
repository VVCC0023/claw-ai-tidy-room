# claw-ai-tidy-room ✨
【零基础友好】AI一键整洁收纳，保持原有家具不变，仅整理杂物+低成本收纳+风格小饰品，支持极简/日系/ins/奶油/电竞/工业风，真实不失真，新手也能快速上手！

---

## 🌟 项目亮点
- **零门槛操作**：无需专业知识，上传照片+发送指令即可完成房间收纳
- **家具零改动**：严格保留原有大件家具、墙面装饰，只优化杂物与收纳布局
- **多风格适配**：支持7种主流风格，从极简到电竞风，满足不同审美需求
- **低成本改造**：仅添加平价小饰品与收纳工具，不涉及硬装改动
- **真实感强**：生成效果贴近现实生活，避免过度美化导致的“效果图”感

---

## 🚀 快速开始（3分钟搞定）

### 1. 准备素材
拍摄你的房间照片（推荐卧室/客厅整体视角），要求：
- 光线充足，无大面积遮挡
- 水平拍摄，避免仰拍/俯拍
- 包含完整家具布局，方便AI识别结构

### 2. 触发指令
发送以下任意指令即可启动整理：
- 基础指令：`帮我整理房间`
- 指定风格：`帮我整理房间，奶油风`
- 查看风格：发送`查看风格`获取完整风格列表

### 3. 等待生成
约10秒即可得到**整洁版效果图**，对比原图直观看到收纳效果：
- 杂物归位，桌面/地面清爽整洁
- 保留所有原有家具与墙面细节
- 自动添加符合风格的低成本小饰品

## ❓ 常见问题

### Q1：生成的图片改变了我的家具怎么办？
A1：本技能默认保留所有大件家具与墙面挂件，若仍有改动，可发送指令：
`保留所有家具 极简风`（替换为你选择的风格）重试

### Q2：图片生成失败？
A2：请尝试以下排查：
1.  检查照片清晰度，避免模糊/黑屏/过度裁剪
2.  确认网络连接正常
3.  更换其他风格尝试生成

### Q3：想调整小饰品数量/风格细节？
A3：可在指令中补充偏好，例如：
- `极简风 少放小饰品`
- `电竞风 多加点RGB灯带`
- `奶油风 增加毛绒抱枕`

## 💡 进阶技巧
1.  **拍照优化**：站在房间中心水平拍摄，确保家具布局完整呈现，提升AI识别准确率
2.  **风格混搭**：可尝试组合风格指令，例如`日系+ins风`，打造个性化效果
3.  **重点标注**：若某区域特别杂乱，可补充说明：
    `我的房间有点乱，重点整理桌面和床面`
4.  **多次迭代**：若对效果不满意，可微调指令重新生成，直到满意为止
